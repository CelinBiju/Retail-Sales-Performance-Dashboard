# Retail Sales Performance Dashboard

## 📌 Project Overview
Developed an interactive Power BI dashboard to analyse retail sales performance, customer trends, category analysis, and regional sales performance.

---

## 🛠 Tools Used
- Power BI
- Excel

---

## 📊 Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Sales by Category
- Sales by Region
- Interactive Filters/Slicers
- Top Products Table
- Business Insights Section

---

## 📈 Key Insights
- Technology category generated highest sales revenue
- West and East regions showed strong performance
- Sales increased over time
- Top-performing products contributed significantly to revenue
